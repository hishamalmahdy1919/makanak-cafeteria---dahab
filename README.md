# Planet Divers — نسخة HTML محسّنة لمحركات البحث (SEO حقيقي)

نسخة موازية طبق الأصل من نفس تصميم ومحتوى تطبيق Flutter، لكن مبنية بـ HTML/CSS/JS ثابت
لتحقيق **فهرسة حقيقية في جوجل** — كل النصوص موجودة فعليًا في كود الصفحة (وليست مُولّدة بجافاسكريبت)،
وكل لغة لها **صفحة/رابط منفصل** (أفضل ممارسة SEO معروفة للمواقع متعددة اللغات).

## 📁 الملفات
```
planet_divers_html/
├── index.html   ← الإنجليزية (الصفحة الافتراضية/الرئيسية)
├── ar.html      ← العربية (RTL تلقائي)
├── ru.html      ← الروسية

├── de.html      ← الألمانية

├── sitemap.xml
└── robots.txt
```

## ✅ ما الذي يجعل هذه النسخة "SEO حقيقي" بخلاف Flutter؟
1. **كل النص HTML ثابت** يظهر فورًا للزائر ولزواحف جوجل، بدون انتظار تحميل جافاسكريبت.
2. **رابط مستقل لكل لغة** (`/ar.html`, `/fr.html`...) بدلاً من تبديل نص بجافاسكريبت — جوجل يفهرس كل لغة على حدة ويعرضها لمستخدمين بلغات مختلفة في بحثهم.
3. **وسوم hreflang** في كل صفحة تخبر جوجل بالنسخ اللغوية البديلة لنفس المحتوى.
4. **JSON-LD (Schema.org)** يوضح لجوجل أن هذا "نشاط رياضي/سياحي" (SportsActivityLocation) مع العنوان والسعر ووسائل التواصل.
5. **sitemap.xml + robots.txt** لتسهيل اكتشاف كل الصفحات وفهرستها.
6. Meta description + keywords مخصصة لكل لغة على حدة (وليست نصًا واحدًا مكررًا).

## 🎥 إضافة فيديوهات يوتيوب (تشتغل ذاتيًا بدون تحميل)
الفيديوهات تُعرض عبر **iframe مباشر من يوتيوب** — أي أنها لا تُحمَّل كملف على السيرفر إطلاقًا،
بل تُبث ("streaming") مباشرة من يوتيوب، وتشتغل تلقائيًا (autoplay) بدون صوت افتراضيًا
(هذا شرط المتصفحات لأي autoplay — يمكن للزائر تشغيل الصوت من عناصر تحكم يوتيوب الظاهرة على الفيديو).

**لإضافة فيديو:** افتح ملف `gen.py` (سكريبت التوليد) وابحث عن هذا السطر:
```python
YOUTUBE_VIDEOS = [
    # "dQw4w9WgXcQ",
    # "L_LUpnjgPso",
]
```
أضف أكواد الفيديوهات (الجزء اللي بعد `v=` في رابط يوتيوب، أو بعد `youtu.be/`) هكذا:
```python
YOUTUBE_VIDEOS = [
    "dQw4w9WgXcQ",
    "L_LUpnjgPso",
]
```
ثم شغّل:
```bash
python3 gen.py
```
سيُعاد توليد كل الصفحات الثمانية تلقائيًا بالفيديوهات الجديدة — بدون الحاجة لتعديل كل صفحة يدويًا.

> إذا مفيش بايثون عندك، تقدر بدل ذلك تنسخ كتلة `<div class="video-card"><iframe ...></iframe></div>`
> يدويًا داخل قسم `id="videos"` في كل ملف HTML على حدة.

## 🖼️ إضافة الصور
ضع صورك في مجلد `images/` بنفس الأسماء المستخدمة في الكود (مثال: `hero-blue-hole-dahab.jpg`)،
أو غيّر اسم الملف في وسم `<img>` داخل `gen.py` (أو مباشرة في ملفات HTML) ليطابق اسم صورتك.
الكود آمن: لو الصورة غير موجودة، تظهر خلفية أنيقة بديلة بدل كسر التصميم.

## 🌐 قبل رفع الموقع فعليًا
غيّر رابط الدومين الحقيقي في `gen.py` (المتغير `SITE_URL`، حاليًا `https://www.planetdivers.com`)
ليكون دومينك الفعلي، ثم شغّل `python3 gen.py` مرة أخرى لتحديث كل روابط hreflang وsitemap.xml وcanonical تلقائيًا.

## 🚀 النشر
الموقع HTML ثابت بالكامل — ما يحتاج أي سيرفر خاص. ارفع كل الملفات كما هي على أي من:
- Netlify / Vercel (اسحب وأفلت المجلد)
- GitHub Pages
- Firebase Hosting
- أي استضافة عادية (cPanel، إلخ) عبر رفع الملفات بالـ FTP

بعد الرفع، سجّل الموقع في **Google Search Console** وأرسل رابط `sitemap.xml` لتسريع الفهرسة.
